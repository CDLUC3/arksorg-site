ARKS - ARK Identifier Resolution Service

ARKS is an ARK identifier resolver that given a provided ARK identifier, matches it to a registered ARK NAAN (prefix) definition. Depending on the form of the request, a successful match will either redirect to the registered target, or present information about the matched definition.

Identifiers take the form:

ark:prefix/value

The service API description is available at /api.

A list of schemes registered with this service is available on the schemes page.

A list of prefixes registered with this service is available on the schemes page.
