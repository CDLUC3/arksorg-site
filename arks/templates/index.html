ARKS - ARK Identifier Resolution Service

ARKS is an ARK identifier resolver that given a provided ARK identifier, matches it to a corresponding identifier definition. Depending on the form of the request, a successful match will either redirect to the registered target or present information about the matched definition.

Identifiers take the form:

ark:prefix/value

The service API description is available at /api.

The registered schemes are listed on the schemes page.
